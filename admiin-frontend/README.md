# Admin Frontend

This is the admin dashboard for ACTA-AI. It includes:
- Sign in page (ID: admin, Pass: admin)
- Dashboard showing API endpoints, user count, meeting count, and analysis data

## Setup
1. Run `npm install`
2. Run `npm run dev` to start the frontend

## Features
- View all API endpoints
- See how many users are present
- See how many meetings are done
- View all analysis data

## Customize
- Update API endpoints in `Dashboard.jsx` as needed
- Connect to your backend for real data
